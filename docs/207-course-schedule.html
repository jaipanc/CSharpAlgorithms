207. Course Schedule

1. Problem Intuition

You are given:

  • numCourses

  • prerequisites

Example:

numCourses = 4 prerequisites = [[1,0],[2,1],[3,2]]

Meaning:

To take course 1 → first take course 0 To take course 2 → first take course 1 To take course 3 → first take course 2

Graph representation:

0 → 1 → 2 → 3

You need to determine:

👉 Can you finish all courses?

The only situation where you cannot finish is if there is a cycle.

Example of cycle:

0 → 1 ↑ ↓ 3 ← 2

If a cycle exists, courses depend on each other forever.

2. Why Topological Sort Works

Topological Sort works only on Directed Acyclic Graphs (DAG).

If we can topologically sort all nodes, then:

No cycle exists → all courses can be finished

If we cannot process all nodes, then:

Cycle exists → cannot finish

Nodes with in-degree 0 can be taken immediately.

5. Algorithm (Kahn's Algorithm)

Steps:

  1. Build adjacency list

  2. Compute in-degrees

  3. Add nodes with in-degree = 0 into queue

  4. Process queue

  5. Reduce neighbors' in-degree

  6. Add new 0 in-degree nodes to queue

  7. Count processed nodes

If:

processed == numCourses

→ return true

Otherwise:

cycle exists

7. C# Implementation

public bool CanFinish(int numCourses, int[][] prerequisites) { List<int>[] graph = new List<int>[numCourses]; int[] indegree = new int[numCourses]; for (int i = 0; i < numCourses; i++) graph[i] = new List<int>(); foreach (var p in prerequisites) { int course = p[0]; int prereq = p[1]; graph[prereq].Add(course); indegree[course]++; } Queue<int> queue = new Queue<int>(); for (int i = 0; i < numCourses; i++) { if (indegree[i] == 0) queue.Enqueue(i); } int completed = 0; while (queue.Count > 0) { int course = queue.Dequeue(); completed++; foreach (var next in graph[course]) { indegree[next]--; if (indegree[next] == 0) queue.Enqueue(next); } } return completed == numCourses; }

8. Complexity

Time Complexity

O(V + E)

  • V = numCourses

  • E = prerequisites

Because we visit every node and edge once.

Space Complexity

O(V + E)

  • adjacency list

  • indegree array

  • queue
