22. Generate Parentheses

Given n pairs of parentheses, write a function to generate all combinations of well-formed parentheses.

Example 1: Input: n = 3 Output: ["((()))","(()())","(())()","()(())","()()()"]
Example 2: Input: n = 1 Output: ["()"]

✅ Step 1: Understand the Rule

A valid parentheses string must:

  1. Never have more ) than ( at any point.

  2. Have exactly n opening ( and n closing ).

✅ Step 2: Visual Example (n = 3)

We need 3 open and 3 close brackets.

Think of this like building the string step-by-step.

We track:

currentString openCount closeCount

🌳 Recursion Tree (Visual)

Start:

"" open=0 close=0

We can add ( because open < 3.

Level 1

"(" open=1 close=0

Level 2

From "("

We can:

  • Add ( (since open < 3)

  • Add ) (since close < open)

"((" open=2 close=0 "()" open=1 close=1

Level 3

From "(("

"(((" open=3 close=0 "(()" open=2 close=1

From "()"

"()(" open=2 close=1

(NOT allowed to add ) because close would exceed open)

Continue Expanding…

Eventually we reach:

((())) (()()) (())() ()(()) ()()()

🎯 Final Answer for n = 3

[ "((()))", "(()())", "(())()", "()(())", "()()()" ]

🧠 Key Backtracking Logic

We follow 2 rules:

if (open < n) → we can add "(" if (close < open) → we can add ")"

When string length == 2 * n → add to result

💻 C# Implementation

public static IList<string> GenerateParenthesis(int n) { var result = new List<string>(); Backtrack(result, "", 0, 0, n); return result; } private static void Backtrack( List<string> result, string current, int open, int close, int n) { // Base case if (current.Length == 2 * n) { result.Add(current); return; } // Add "(" if (open < n) { Backtrack(result, current + "(", open + 1, close, n); } // Add ")" if (close < open) { Backtrack(result, current + ")", open, close + 1, n); } }

⏱ Complexity

Time Complexity: O(4ⁿ / √n) (Catalan number growth)

Space Complexity: O(n) recursion depth

🧪 xUnit Test using TheoryData

We will use Assert.Equivalent to compare collections.

using Xunit; using System.Collections.Generic; public class GenerateParenthesesTests { public static TheoryData<int, IList<string>> TestData => new() { { 1, new List<string> { "()" } }, { 2, new List<string> { "(())", "()()" } }, { 3, new List<string> { "((()))", "(()())", "(())()", "()(())", "()()()" } } }; [Theory] [MemberData(nameof(TestData))] public void GenerateParenthesis_ShouldReturnExpected(int n, IList<string> expected) { var result = GenerateParenthesis(n); Assert.Equivalent(expected, result); } private static IList<string> GenerateParenthesis(int n) { var result = new List<string>(); Backtrack(result, "", 0, 0, n); return result; } private static void Backtrack( List<string> result, string current, int open, int close, int n) { if (current.Length == 2 * n) { result.Add(current); return; } if (open < n) Backtrack(result, current + "(", open + 1, close, n); if (close < open) Backtrack(result, current + ")", open, close + 1, n); } }
